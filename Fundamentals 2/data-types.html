Boolean It is a primitive data type in JS. By primitive it means that it can hold only one value at a time. Booleans are consisted of values "true" or "false". It is useful to use it in conditional statements. The source
Null It is a primitive value that represents an intentional absence of any object value. Null means that it points to no object. The source
Undefined It is a primitive data type that has only one value undefined. It shows when the variable is declared but no value is assigned to it. The source
Number It is a primitive data type that is used to represent positive or negative numbers. It can be integer, floating-point numbers and numbers with scientific notations. The source
BigInt It is numberic primitive type that represent large numbers. It is used with n in the end of the number. BigInt can not be calculated with Number value. It should be calculaated only to the same type. The source
String It is a data type that represents textual data surrounded by single or double quotes or by backticks. Strings are immutable which means that they can not be modified. The source
Symbol It is a primitive data type. This type is used to create unique identifiers for objects. The source
